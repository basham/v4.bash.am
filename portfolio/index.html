Course Search proof of concept

March – May 2013 · Video demo · GitHub repository

Concept

Build a proof of concept web application for students to search and bookmark classes.

Process

This 6-week project was a collaboration between the Process Experience Architecture team and the Student Information Services team within Indiana University’s University Information Technology Services department. It was the first attempt within UITS to pair a client-side framework (AngularJS) with RESTful data services for a student-facing application.

Credits

  • Designed by Chris Basham and Tara Bazler.
  • Front-end developed by Chris Basham.
  • Back-end API developed by Mark Fyffe.
